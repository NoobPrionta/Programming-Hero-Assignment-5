1.Difference between var, let, and const.

Ans :var – Old way to declare variables. Can be redeclared and updated.
let – Modern variable. Can be updated but not redeclared in the same scope.
const – Used for values that should not change. Cannot be updated or redeclared.

2.What is the Spread Operator (...)?

Ans:The spread operator (...) is used to copy or expand arrays and objects.

3.Difference between map(), filter(), and forEach().

Ans: map() – Creates a new array by modifying each element.
filter() – Creates a new array with elements that match a condition.
forEach() – Runs a function for each element but does not return a new array.

4.What is an Arrow Function?

Ans:An arrow function is a shorter way to write functions in JavaScript.

5.What are Template Literals?

Ans:Template literals allow variables to be included inside strings using backticks.
